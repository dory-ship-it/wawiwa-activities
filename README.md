# Wawiwa Activities

Public, student-facing interactive class activities for Wawiwa programs.
Each activity is a self-contained HTML page (no external requests, no data
collection) opened by students in a new browser tab from inside the gated
course. Students fill it in and use their browser's Save-as-PDF to keep a copy.

## Activities
- `eai-m1l1-workbook/` — Enterprise AI Implementation, Module 1 Lesson 1:
  "Diagnosing Use Cases" class workbook.

Live URL pattern: https://<your-github-username>.github.io/wawiwa-activities/eai-m1l1-workbook/
